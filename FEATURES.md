# ✨ 功能特性详解

## 🎨 视觉设计

### 手绘风格
- **Sketch Cards**: 带有手绘边框和阴影效果的卡片
- **Rough Notation**: 手写风格的下划线、圆圈和高亮效果
- **自定义光标**: 粉色圆点光标，按钮悬停时变为紫色
- **手写字体**: 使用 Caveat 字体增添个性

### 杜鹃花主题 🌸
- **SVG 手绘花朵**: 完全自定义的杜鹃花组件
- **漂浮动画**: 花朵在页面中优雅地漂浮
- **装饰元素**: 在各个部分点缀杜鹃花装饰
- **主题色**: 粉色系配色方案

## 🎯 交互功能

### 1. 交互式粒子背景
**位置**: 全局背景
**功能**:
- 50 个彩色粒子在画布上移动
- 粒子之间自动连接形成网络
- 鼠标靠近时粒子会躲避
- 粉色系渐变色彩

**技术**: Canvas API + 实时动画

### 2. 鼠标轨迹效果
**位置**: 全局
**功能**:
- 跟随鼠标移动的彩色轨迹
- 渐变透明度和大小
- 可在页脚开关
- 不影响页面交互

**技术**: React Hooks + 动态 DOM 元素

### 3. 涂鸦板
**位置**: 右下角浮动按钮
**功能**:
- 多种颜色选择
- 可调节画笔大小
- 橡皮擦工具
- 清空画布
- 下载作品为 PNG

**技术**: Canvas API + 事件处理

### 4. 视差效果
**位置**: Hero 部分
**功能**:
- 主内容随鼠标移动产生视差
- 平滑过渡动画
- 增强沉浸感

**技术**: 鼠标位置追踪 + CSS Transform

### 5. 互动表情
**位置**: Hero 部分顶部
**功能**:
- 6 个表情图标
- 悬停时放大和旋转
- 点击时全屏显示
- 弹跳动画

**技术**: CSS 动画 + React 状态管理

## 🎭 动画效果

### 页面加载动画
- **fadeInUp**: 元素从下方淡入

- **bounce**: 弹跳效果

- **wiggle**: 摇摆效果

### 悬停效果
- **卡片**: 平移 + 阴影增强
- **按钮**: 缩放 + 旋转
- **链接**: 下划线动画
- **图标**: 旋转 + 缩放

### Rough Notation 动画
- **underline**: 手绘下划线
- **circle**: 手绘圆圈
- **box**: 手绘方框
- **highlight**: 高亮标记
